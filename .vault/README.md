# Memory Vault

This Obsidian vault stores durable agent memories for this repository.

- Pending memories: `memories/`
- Archived memories: `archives/`
- Template: `.opencode/plugin/memory-template.md`
- Promotion command: `/apply-memories`

Pending memory notes are generated by `.opencode/plugin/auto-memory.ts`. Run `/apply-memories` in opencode to promote useful lessons into `AGENTS.md` files or agent skills and archive processed notes.
