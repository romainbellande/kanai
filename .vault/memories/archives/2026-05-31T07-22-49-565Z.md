# Memory - Memories To Skills Command

## Insight
Kanai now has an opencode slash command at `.opencode/commands/memories-to-skills.md` that converts active top-level `.vault/memories/` notes into new or updated project skills, excludes `memories/archives/` from input, and archives processed memories via Obsidian MCP moves.
