# Memory - Opencode Memory Plugin

## Insight
Kanai now has a local opencode memory plugin at `.opencode/plugins/memory.ts`. It injects instructions that require future agents to use the `obsidian` skill and the project Obsidian MCP server to write timestamped memory notes in `memories/`.

## Use When
When future agents need to persist durable lessons, inspect or maintain the memory workflow, or understand why `.vault/memories/` contains opencode memory notes.
