# Memory - TanStack Unnested Project Routes

## Insight
In Kanai's TanStack Router file routes, a task form at `/projects/$projectId/tasks/new` must use the trailing-underscore escape (`src/routes/projects_.$projectId.tasks.new.tsx`) to avoid being nested under `src/routes/projects/$projectId.tsx`; otherwise the board route remains the parent and needs an `Outlet` to render the child.

## Use When
When adding project-scoped pages that should replace the board view rather than render inside it, use an unnested file-route name and `useParams({ from: "/projects_/$projectId/tasks/new" })` while linking with `to="/projects/$projectId/tasks/new"`.
